Instructions: guided manual scoring via random intratumoral circles

  1. Please scan the slide of interest using Aperio scanner
  2. Please use Aperio ImageScope to open the virtual slide file (*.svs).
    • Mark/select the tumor area using annotation pen tool, rectangle tool or ellipse tool.
    • Please note: negative pen tool NOT supported
    • Please save the annotation file (using the default name)
  3. This application will then generate five intratumoral circles (virtual TMA cores), randomly distributed among the selected tumor area(s). Please count 100 tumor nuclei from each of the five intratumoral circles (i.e. a total of 500 nuclei for five intratumoral circles). Please note: if during the scoring if at least one of the intratumoral circles has less than 100 tumor nuclei, please count all tumor nuclei on the current circle and proceed to the next circle. You can safely ignore the message to ask you to score from the top/bottom of the circle since all tumor nuclei on this particular circle will be counted. Additional random intratumoral circle(s) will be generated until at least 500 nuclei is counted.
  4. This application will then assess the heterogeneity of the Ki67 expression based on the scores (i.e. positive/negative nuclei count) on the five intratumoral circles. Sufficiently high heterogeneity will trigger (up to 20) additional intratumoral circles being generated and presented for scoring. Each core would require 100 tumor nuclei being counted in the same fashion as the initial five.
  5. System requirements
    • Windows XP or higher
    • Aperio ImageScope (version 12 or higher)
    • Minimum screen resolution: 1024 x 768
  6. Please verify all system requirements and perform all tasks mentioned above. When you are ready, please click on the "OK" button below to continue. You will be asked to select the Aperio virtual slide file (*.svs).
